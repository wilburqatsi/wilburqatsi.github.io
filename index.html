Village of Point Venture

A professional city website rebuilt and redesigned for the Village of Point Venture, TX. The website is built on the WordPress platform. I created a custom mobile-responsive WordPress theme that shows off the cool lakeside attitude of Point Venture.

Amazing Babies!

Amazing Babies! is a web scraping app that presents random exotic cat facts to the user. The app uses the Python Beautiful Soup web-scraping library to grab a single fact from over 40 exotic feline species on Wikipedia. Witness gorgeous cats and educate yourself with the click of a single button. This app is built on the Python-Flask framework. Front-end is created using HTML/CSS/Javascript and Bootstrap.

To Do List

To-do List is a web-app that helps you keep track of the things you need to do. Click a button anytime you have a new responsibility. Edit it if you make a mistake. Delete it when you're done with it or when you want to procrastinate. Created using HTML/CSS/Javascript.
