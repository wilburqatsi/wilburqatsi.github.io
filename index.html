Amazing Babies!

Amazing Babies! is a web-app that presents random exotic cat facts to the user. The app uses the Python Beautiful Soup web-scraping library to grab a single fact from over 40 exotic feline species. Witness gorgeous cats and educate yourself with the click of a single button. This app is built on the Python-Flask framework. Front-end is created using HTML/CSS/Javascript and Bootstrap.

To Do List

To-do List is a web-app that helps you keep track of the things you need to do. Click a button anytime you have a new responsibility. Edit it if you make a mistake. Delete it when you're done with it or when you want to procrastinate. Created using HTML/CSS/Javascript.
